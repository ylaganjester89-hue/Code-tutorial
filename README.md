A web-based platform to practice coding across **500+ levels** in **HTML, CSS, JavaScript, Python, Java**, and more. Features include:  
  
- 3 Tiers: Simple (500), Intermediate (700), Master (1000) levels  
- Live code editor to test HTML/JS/Python directly in browser  
- Skippable levels with partial points  
- Dynamic leaderboard (top 10 users)  
- AI hints for each level  
  
## How to Use  
  
1. Open `index.html` in your browser.  
2. Select a tier from the sidebar.  
3. Type your code in the editor and run it.  
4. Complete or skip levels to earn points.  
5. Check the leaderboard for top scores.
